# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.1.0] - 2026-02-10

### Added
- `stats.rangeListenCount` in the output JSON, computed from the ListenBrainz stats listening-activity endpoint for the selected `stats_range`

## [2.0.0] - 2026-02-10

### Changed
- **BREAKING**: Repository renamed from `listenbrainz-github-action` to `listenbrainz-json-bourne`
- Action name updated to "ListenBrainz JSON Bourne"
- Updated action description for better marketplace discoverability

### Added
- Logo and branding assets
- Social media preview images

### Note
This is the initial public release under the new name. No functional changes from v1.1.3.
